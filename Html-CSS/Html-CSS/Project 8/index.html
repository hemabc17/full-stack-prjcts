Introduction

MySQL is an open-source relational database management system used to store and manage data.MySQL is an open-source relational database management system used to store, organize, and manage data efficiently.It uses Structured Query Language (SQL) to perform operations like creating databases, inserting data, and retrieving information.MySQL is widely used in web applications and works well with languages like PHP and JavaScript.

Installation

Download MySQL from the official website and install it on your system.

Create Database

CREATE DATABASE college;

Create Table

CREATE TABLE students (
   id INT PRIMARY KEY,
   name VARCHAR(50),
   age INT
);

SELECT Query

SELECT * FROM students;

Constraints

  • PRIMARY KEY
  • FOREIGN KEY
  • NOT NULL
  • UNIQUE

JOIN

SELECT students.name, courses.course_name
FROM students
INNER JOIN courses
ON students.id = courses.student_id;
